Creating a Server

To create a server, go to the client page, click Create Server, which you can find on the left sidebar.

Give your server a name, put in the amount of ram you choose to use(1000MB or lower is fine), select a location that is nearby for you and your visitors.

Once you have clicked create, your server start being created. This may take between 5 â€“ 10 minutes, if it takes longer, then delete and try again.

Then head over to the panel and select your server. On the console, click start. Select 9 for Others and then 8 for Nginx.

After selecting Nginx, the server will download required files, stop, then start again. Now your web server is running. Make sure your web files go into /webroot/.

Extra Information:

Nginx is an HTTP and reverse proxy server, a mail proxy server, and a generic TCP/UDP proxy server, originally written by Igor Sysoev.
