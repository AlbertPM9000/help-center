Connecting your Domain

When connecting your domain to a web server on FalixNodes, you’ll the numberic IP address and port number. You’re also required to install an app on the domain in Cloudflare, which will be told later.

In the DNS settings, create an A Record, set “@” as the name and the numberic IP as IPv4 address.

We need a way to tell Cloudflare to use a different port number, making sure the domain uses the port FalixNodes provides to the user. By default, Cloudflare will try to use port 80.
Click on Works and click on Manage Workers. In Works, click on Create Worker and remove all code that is provided by Cloudflare.

Use this code instead:

addEventListener('fetch', event => {
                  const request = event.request;
                  var url = new URL(request.url);
                  url.host = 'my.domain.com:12345';
                  event.respondWith(fetch(url, request));
                });

On the forth line, change “my.domain.com” to your domain and switch the port “12345” to the port FalixNodes provides you.
