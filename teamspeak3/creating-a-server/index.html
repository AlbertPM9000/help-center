Creating a Server

Creating a TeamSpeak3 server is easy to do.

Simply go to the client and click on "Create a Server" found in the sidebar. Go ahead fill out the information and make sure to select TeamSpeak3 at the bottom, then click create.

Once server has been created in the client, go to your panel and select your TeamSpeak3 server. In the console, click "Start", you'll be asked to choose some numbers which will be 4(Voice) then 1(TeamSpeak3). Server should start booting, you'll also be provided an auth key, which you will need when logging into the server for the first time(Keep the auth key private). Log into your TeamSpeak3 server in TeamSpeak, then use the auth key after clicking connect.

Enjoy your TeamSpeak3 server.
