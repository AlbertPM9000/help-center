When logging in with Discord, a bunch of servers show up on my server list, why is this?

This occurs when using logging in with offers, by joining these extra servers, you'll recieve more resources. If you did not choose the offers option when logging in via Discord, then there must of been a mistake on our end.

How do I fix the error "We're having some trouble connecting to your server, please wait..."

This error means that the wings are not working for your node. The wings are the connection between the panel and the node. If the wings are offline your server will be not offline, only the panel will not work properly. There is no way to fix this, the wings automatically restart every 20 minutes. If you wait a few minutes it should be fixed.

How do I upload my world?

It is not possible to drag and drop a folder. If you need to upload a folder from your pc to the server, then use SFTP.

Error Event [xxx.xxx.xxx]: another power action is currently being processed for this server, please try again later

This error you get when too many people try to start their server. The connection cannot handle so many requests so it gives an error. If the wings are overloaded your server will be not offline, only the panel will not work properly. Most time this error will fix when you try it later again, do not spam click any start/stop button.

How do I remove the java version selection

If you want your server to automatically choose your server version to follow these steps;

  • Open your server in the Game Panel

  • Go to the startup tab
  • Find the JAVA VERSION section

    • Use one of these following options:

  • adopt@1.11.0-9
    if you are using 1.13+

  • version adopt@1.8.0-275
