FalixNodes Help Center

Uninstalling

This article has been updated for FalixNodes Software v2.2.0

There are reasons to why you might want to clear cache, it may also help the app to run better if you clear cache every month or two.

Clearing Cache on Windows:

%appdata%\FalixNodes

On Windows, you can type that path in to Run(Start + R) and it will bring you there.

Clearing Cache on macOS:

Open Finder and in the menubar, click on Go and Go To Folder, use this path:

~/Library/Application Support/FalixNodes/

Drag the file named “Cookie” into your trash bin.

Clearing Cache on Linux:

Deleting cache manually can break Snap, so we need to remove FalixNodes with Snap then re-install it:

sudo snap remove --purge falixnodes
                sudo snap install falixnodes --beta

Clearing Cache on iOS:

Go to Settings > Safari > then tab on Clear History and Website Data.
The app for iOS relies on Safari, this is data has to be cleared in Safari in order to clear cache on the app.
