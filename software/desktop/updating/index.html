FalixNodes Help Center

Uninstalling

If you need to for any reasons like freeing up space, you can uninstall the app.

Windows

If you see that it's downloading a new update, keep the software open until a notification appears saying it's ready. When it's ready, close the software and it the update will start installing.

Linux

Snapcraft will auto update the app in the background, or if you want to force update it, you can run this:

sudo snap refresh falixnodes
