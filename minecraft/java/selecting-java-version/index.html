Selecting Java Version

When starting up a Minecraft Java server, you're ask to what version of Java you want to use. Please use Java 8 when using Minecraft 1.16.4 or older, or use Java 11 for Minecraft 1.17 or newer.

To skip this part of starting a Minecraft Java server, head over to the startup settings and set of these for JAVA VERSION:

  • adopt@1.8.0-275
  • adopt@1.11.0-9
