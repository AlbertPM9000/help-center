Creating a Minecraft Server

Creating the Server

To create a server, go to the client page, click Create Server, which you can find on the left sidebar.

Give your server a name, put in the amount of ram you choose to use(2000MB recommended), select a location that is nearby for you and your community, and lastly select the type of server you want which might be Minecraft Java or Minecraft Bedrock.

Once you have clicked create, your server start being created. This may take between 5 - 10 minutes, if it takes longer, then delete and try again.

Setting it up

In your control panel, go to the console of the new server you created, then click start. This is where you select the type and version for your server. So if you want to create a vanilla server, you would type "7" in; or if you want to create a PaperSpigot server, you would type "1" in. Once you have selected the type of server you want, choose the version you would like, you will type the version number in, something like "1.8.9" or "1.16.1". After that, FalixNodes will download what's required and then boot the server, generating everything needed.
