File Manager

The file manager is where you can edit, copy, delete, and upload to your server's files.

Selecting a file/files and folder/folders

To select a file or a folder, you click on the little box that is on the left side of it's name, which is a checkbox.

Deleting a File/Folder

Select the file you wish to delete, then up top right, click on "Mass Actions". Click "Delete".

Editing a File

Simply click exactly on the file's name. An editor will show.
