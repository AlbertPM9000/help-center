Changing Server Name

Don't like the name you choice or maybe you're doing some rebranding? You can easily change the name of your server, here's how to do it.

In your panel, click on the name of the server you want to change. On the sidebar, open the "Configuration" drop down menu and click "Server Name". In the text box already containing your server's name, change the name to waht you want, and click Submit.
