Error Messages

Need help understanding an error message and what could be wrong, this should help.


Errors in Client:

No nodes are satisfying the requirements specificed for automatic deployment could be found.

Server creation has been disabled for selected node.

There was an exception while attempting to communicate with the daemon resulting in a HTTP/E_CONN_REFUSED response code. This exception has been logged.

Unknown

Errors in Panel:

There was an error attempting to establish a WebSocket connection to the Daemon. This panel will not work as expected.

This is simply a timeout error. When this error appears in the panel, just wait a few moment and eventfully it will go away.
